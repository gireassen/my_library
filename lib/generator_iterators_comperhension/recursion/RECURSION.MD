# Recursion

Рекурсия - вызов функцией самой себя.

Рекурсивные функции занимают больше места в памяти по сравнению с итеративными из-за постоянного добавления новых слоев в стек в памяти. Однако их производительность куда выше.

>**Рекурсия может быть медленной, если реализована неправильно.**

*Пример рекурсии:*

```
def look_for_key(box):
    for item in box:
        if item.is_a_box():
            look_for_key(item):
        elif item.is_a_key():
            print("found the key")
```

# Базовый и рекурсивный случай

Можно легко ошибиться и сделать цикл бесконечным:

```
def countdown(i):
    print(i)
    countdown(i-1)
```

Когда пишем рекурсивную, необходимо указать, в какой момент следует прервать рекурсию.
Каждая рекурсивная функция состоит из двух частей: *базового случая и рекурсивного случая.*

Добавим базовый случай в функцию **countdown**:

```
def countdown(i):
    print(i)
    if i <= 1:
        return
    else:
        countdown(i-1)
```
